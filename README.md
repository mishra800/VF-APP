## ğŸš€ **Getting Started**

### **Prerequisites**
- Node.js (>= 20)
- React Native CLI
- Android Studio (for Android)
- Xcode (for iOS)

### **Installation**

1. **Clone the repository**
```bash
git clone <repository-url>
cd CarSeizureApp
```

2. **Install dependencies**
```bash
npm install
```

3. **iOS Setup** (iOS only)
```bash
cd ios
bundle install
bundle exec pod install
cd ..
```

### **Running the App**

1. **Start Metro**
```bash
npm start
```

2. **Run on Android**
```bash
npm run android
```

3. **Run on iOS**
```bash
npm run ios
```
